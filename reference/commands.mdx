title: "TShock in-game command reference"
description: "Complete reference for TShock's in-game commands covering server administration, player management, world control, NPC spawning, teleportation, and more."
---

TShock extends Terraria's server with a rich command system accessible in-game using the `/` prefix or silently with `.`. Every command is gated by a permission node, making it easy to grant fine-grained control to players, moderators, and administrators. This page documents every built-in command registered by TShock, organized by category, including required permissions and usage syntax.

<Note>
  Commands marked **in-game only** cannot be run from the server console. The silent specifier `.` runs a command without broadcasting it to players with log permissions.
</Note>

---

## Setup

| Command | Permission | Description | Usage |
|---------|------------|-------------|-------|
| `/setup` | *(none — token-based)* | Authenticates as superadmin during first-time TShock setup. In-game only. | `/setup <token>` |

## Account Commands

Commands for managing player authentication and registration.

| Command | Aliases | Permission | Description | Usage |
|---------|---------|------------|-------------|-------|
| `/login` | — | `tshock.account.login` | Logs you into a registered account. In-game only. | `/login <password>` |
| `/logout` | — | `tshock.account.logout` | Logs you out of your current account. In-game only. | `/logout` |
| `/password` | — | `tshock.account.changepassword` | Changes your account's password. In-game only. | `/password <old> <new>` |
| `/register` | — | `tshock.account.register` | Registers a new account. In-game only. | `/register <password>` |
| `/accountinfo` | `/ai` | `tshock.accountinfo.check` | Shows registration and last-login info for a username. | `/accountinfo <username>` |
| `/user` | — | `tshock.superadmin.user` | Manages user accounts (superadmin). | `/user <add\|del\|group\|password> <args>` |

| `/tileban` | — | `tshock.admin.tileban` | Manages the tile ban list. | `/tileban <add\|del\|list> <id>` |

| `/uploadssc` | — | `tshock.ssc.upload` | Uploads your character data at join time as SSC data. | `/uploadssc` |
| `/tempgroup` | — | `tshock.admin.tempgroup` | Temporarily sets another player's group. | `/tempgroup <player> <group>` |
| `/su` | — | `tshock.su` | Temporarily elevates you to Super Admin for 10 minutes. | `/su` |
| `/sudo` | — | `tshock.su` | Executes a command as Super Admin. | `/sudo <command>` |
| `/userinfo` | `/ui` | `tshock.admin.userinfo` | Shows account and connection info for a player. | `/userinfo <player>` |

| Command | Aliases | Permission | Description | Usage |
|---------|---------|------------|-------------|-------|
