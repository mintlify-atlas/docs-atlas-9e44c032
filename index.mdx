## What TShock provides

TShock is a plugin that runs on the Terraria Server API (TSAPI), adding a comprehensive management layer on top of the vanilla Terraria server.

<CardGroup cols={2}>
  <Card title="Anti-cheat & Protection" icon="lock">
    Block item spawning, tile manipulation exploits, and other client-side cheats with built-in detection systems.
  </Card>
  <Card title="Server-Side Characters" icon="user">
    Store player inventories, health, and progression server-side so characters can't be modified outside your server.
  </Card>
  <Card title="Groups & Permissions" icon="users">
    Create hierarchical permission groups to control exactly what each player can do — from chatting to world editing.
  </Card>
  <Card title="REST API" icon="globe">
    Manage your server remotely with a built-in HTTP REST API for bans, players, groups, and more.
  </Card>
  <Card title="Ban System" icon="ban">
    Ban players by IP address, UUID, or account name with optional time-limited bans and reasons.
  </Card>
  <Card title="NuGet Plugin Manager" icon="box">
    Install and manage TShock plugins directly from NuGet repositories with automatic dependency resolution.
  </Card>
</CardGroup>

## Get started in three steps

<Steps>
  <Step title="Download and install TShock">
    Download the latest release from GitHub, extract it, and run `TShock.Server` for your platform.
  </Step>
  <Step title="Complete initial setup">
    Use the `/setup` command in-game with the code shown in your server console to gain temporary admin access.
  </Step>
  <Step title="Create your admin account">
    Run `/user add <name> <password> owner` to create a permanent admin account, then `/login` to authenticate.
  </Step>
</Steps>

<Note>
  TShock requires .NET 9 and supports Windows, Linux (x64, arm64, arm32), and macOS. At least 1 GB of RAM is recommended for a stable server.
</Note>
